# Rspack 特有属性

Loader API 提供了所有模块相关的信息。然而，在极少数情况下，你可能需要访问编译器 API 本身。

:::warning 警告
请注意，使用这些特定于 Rspack 的属性会对你的 loader 的兼容性产生负面影响。

因此，你应该将它们作为最后的手段使用。使用它们会降低你的 loader 的可移植性。
:::

## this.\_compilation

访问 Rspack 当前的 Compilation 对象。

## this.\_compiler

访问 Rspack 当前的 Compiler 对象。
