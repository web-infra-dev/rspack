# ElectronTargetPlugin

<ApiMeta addedVersion={'0.3.3'} />

此插件用来在打包时将 Electron 的内置模块外置掉，[`externalsPresets.electron`](/config/externals#electron)、[`externalsPresets.electronMain`](/config/externals#electronmain)、[`externalsPresets.electronRenderer`](/config/externals#electronrenderer) 和 [`externalsPresets.electronPreload`](/config/externals#electronpreload) 底层使用了该插件。

```js
new rspack.electron.ElectronTargetPlugin();
```
