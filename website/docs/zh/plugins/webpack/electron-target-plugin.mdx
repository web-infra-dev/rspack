# ElectronTargetPlugin

这个插件用于在打包时 external Electron 内置模块，被 [`externalsPresets.electron`](/config/externals#electron)、[`externalsPresets.electronMain`](/config/externals#electronmain)、[`externalsPresets.electronRenderer`](/config/externals#electronrenderer) 和 [`externalsPresets.electronPreload`](/config/externals#electronpreload) 所使用。

```js
new rspack.electron.ElectronTargetPlugin();
```
