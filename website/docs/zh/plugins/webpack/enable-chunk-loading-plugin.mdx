# javascript.EnableChunkLoadingPlugin

<ApiMeta addedVersion={'0.3.4'} />

开启该 chunkLoadingType 的运行时模块打包，[output.enabledChunkLoadingTypes](/zh/config/output.html#outputenabledchunkloadingtypes) 底层使用了该插件。

```js
new rspack.javascript.EnableChunkLoadingPlugin('import');
```
