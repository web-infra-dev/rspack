# ProgressPlugin

<ApiMeta addedVersion={'0.3.3'} />

此插件可以用来配置进度条。

```js
new rspack.ProgressPlugin(options);
```

## 选项

- **Type:** `{ prefix?: string }`
- **Default:** `{}`

### prefix

- **Type:** `string`
- **Default:** `'Rspack'`

进度条前显示文案。
