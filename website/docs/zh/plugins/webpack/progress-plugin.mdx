# ProgressPlugin

<ApiMeta addedVersion={'0.3.3'} />

此插件可以用来配置进度条。

```js
new rspack.ProgressPlugin(options);
```

- options
  - **类型：** `{ prefix?: string }`
  - **默认值：** `{}`
