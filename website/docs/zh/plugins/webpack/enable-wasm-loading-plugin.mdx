# wasm.EnableWasmLoadingPlugin

<ApiMeta addedVersion={'0.3.3'} />

开启该 wasmLoadingType 的运行时模块打包，[output.enabledWasmLoadingTypes](/config/output#outputenabledwasmloadingtypes) 底层使用了该插件。

```js
new rspack.library.EnableWasmLoadingPlugin('fetch');
```
