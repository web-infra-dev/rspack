# EntryPlugin

<ApiMeta addedVersion={'0.3.3'} />

在编译时添加一个入口 chunk。该 chunk 的名称为 `options.name`，仅包含一个模块（以及依赖项）。该模块是通过 `entry` 在 `context` 下（绝对路径）解析出来的。

```js
new rspack.EntryPlugin(context, entry, options);
```

## 选项

### context

入口模块的请求路径会在该 `context` 路径下解析为绝对路径。

- **类型：** `string`

### entry

入口模块的请求路径。

- **类型：** `string`

### options

调整入口模块的相关设置。

- **类型：**
```ts
type EntryOptions = {
  name?: string;
  runtime?: EntryRuntime;
  chunkLoading?: ChunkLoading;
  asyncChunks?: boolean;
  publicPath?: PublicPath;
  baseUri?: string;
  filename?: Filename;
};
```
