# 品牌指南

在这里你可以找到 Rspack 的品牌指南、资源和许可证。

## 名称

Rspack 在独立使用的情况下都应该写作 Rspack，而不是 rspack 或者 RSPACK，但在命令行或者包名中可以使用小写，如 `@rspack/cli`。

## Logo

Rspack 的 Logo 是一个如闪电一样迅速的小螃蟹，代表了 Rspack 的快速编译。
