# 迁移 Create React App

由于 Create React App（CRA）内置了丰富的工程能力，如果使用 Rspack CLI 手动搭建一个具有同等能力的配置会比较困难。因此，我们推荐将 CRA 应用迁移到 [Rsbuild](https://rsbuild.dev/)，以便开箱即用地使用 Rspack 的能力。

## 什么是 Rsbuild

Rsbuild 是由 Rspack 驱动的高性能构建工具，它默认包含了一套精心设计的构建配置，提供开箱即用的开发体验，并能够充分发挥出 Rspack 的性能优势。

Rsbuild 提供丰富的构建功能，包括编译 TypeScript、JSX、Sass、Less、CSS Modules、Wasm，以及其他资源，也支持模块联邦、图片压缩、类型检查、PostCSS、Lightning CSS 等功能。

![build tools](https://assets.rspack.dev/rsbuild/assets/rsbuild-1-0-build-tools.png)

## 如何迁移

Rsbuild 提供了一个全面的迁移指南，帮助你从 Create React App 迁移。

请参考 [迁移指南](https://rsbuild.dev/zh/guide/migration/cra) 来了解更多内容。
