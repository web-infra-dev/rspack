# Cache

缓存：该选项可以开启或者关闭 Rspack 构建过程中对快照及中间产物的缓存，如果开启，在下次构建中可以使用它们来提升构建的速度。

:::info 缓存类型
Rspack 当前支持内存缓存（Memory Cache）和实验性的持久化缓存（Persistent Cache），更多细节参考 [Persistent Cache](/config/experiments#持久化缓存)。
:::

此配置当前仅支持控制是否进行缓存。

<PropertyType
  type="boolean"
  defaultValueList={[
    { defaultValue: 'false', mode: 'production' },
    { defaultValue: 'true', mode: 'development' },
  ]}
/>

## 用法

你可以在 `rspack.config.js` 中直接设置 cache 来禁用缓存:

```ts title="rspack.config.js"
module.exports = {
  cache: false,
};
```
