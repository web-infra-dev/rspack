# Cache

Rspack will cache snapshots and intermediate products during the build process and use them in the next build to improve the speed of the build.

:::info Cache Type
Rspack currently supports the memory cache and the experimental persistent cache, see [Persistent Cache](/config/experiments#persistent-cache) for more details.
:::

This configuration currently only supports setting whether to enable caching.

<PropertyType
  type="boolean"
  defaultValueList={[
    { defaultValue: 'false', mode: 'production' },
    { defaultValue: 'true', mode: 'development' },
  ]}
/>

## Usage

You can disable the `cache` directly in Rspack config:

```js title="rspack.config.mjs"
export default {
  cache: false,
};
```
