# ProgressPlugin

<ApiMeta addedVersion={'0.3.3'} />

This plugin can be used to configure the progress bar.

```js
new rspack.ProgressPlugin(options);
```

## Options

- **Type:** `{ prefix?: string }`
- **Default:** `{}`

### prefix

- **Type:** `string`
- **Default:** `'Rspack'`

The text will be displayed before the progress bar.
