# javascript.EnableChunkLoadingPlugin

<ApiMeta addedVersion={'0.3.4'} />

Enable runtime module bundling for this chunkLoadingType, and is used by [output.enabledChunkLoadingTypes](/en/config/output.html#outputenabledchunkloadingtypes) under the hood.

```js
new rspack.javascript.EnableChunkLoadingPlugin('import');
```
